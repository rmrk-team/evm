# IRMRKCore

*RMRK team*

> IRMRKCore

Interface smart contract for RMRK core module.



## Methods

### name

```solidity
function name() external view returns (string)
```

Used to retrieve the collection name.




#### Returns

| Name | Type | Description |
|---|---|---|
| _0 | string | string Name of the collection |


#### Returns

| Name | Type | Description |
